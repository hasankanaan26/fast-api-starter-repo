# Task Assistant API

A simple REST API for managing tasks (to-do items), built with **FastAPI** and **Python**.  
You can create, read, update, delete, and search tasks — all through HTTP requests.

> **What is an API?** An API (Application Programming Interface) is a way for programs to talk to each other. This project creates a web server that listens for HTTP requests (like the ones your browser makes) and responds with JSON data. Think of it like a waiter in a restaurant — you tell the waiter what you want, and they bring it back from the kitchen.

This project uses **[uv](https://docs.astral.sh/uv/)** to manage Python versions, dependencies, and virtual environments. uv is a fast, modern replacement for pip, pip-tools, and virtualenv — all in one tool.

---

## Quick Start

```bash
# 1. Clone the project
git clone https://github.com/YOUR_USERNAME/TaskAssistant.git
cd TaskAssistant

# 2. Install everything (Python, packages, virtual environment — all in one command)
uv sync

# 3. Start the server
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Your API is now running at **http://localhost:8000**.  
Interactive docs are at **http://localhost:8000/docs** (Swagger UI).

> **Don't have uv?** It's a single binary — no Python needed to install it.  
> macOS/Linux: `curl -LsSf https://astral.sh/uv/install.sh | sh`  
> Windows: `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`

## Run With Docker

```bash
# Build the image
docker build -t task-assistant .

# Run the container
docker run -d -p 8000:8000 --name task-assistant-app task-assistant
```

No Python or uv needed on your machine — Docker handles everything.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/health` | Health check — returns `{"status": "ok"}` |
| `GET` | `/tasks` | List all tasks |
| `POST` | `/tasks` | Create a new task |
| `GET` | `/tasks/{task_id}` | Get a specific task by ID |
| `GET` | `/tasks/search/?q=query` | Search tasks by title or description |
| `PUT` | `/tasks/{task_id}` | Update a task (partial updates supported) |
| `DELETE` | `/tasks/{task_id}` | Delete a task |

Quick test:
```bash
# Create a task
curl -X POST http://localhost:8000/tasks \
  -H "Content-Type: application/json" \
  -d '{"title": "Learn FastAPI", "description": "Build my first API project"}'

# List all tasks
curl http://localhost:8000/tasks
```

See the full reference with all examples in [docs/API.md](docs/API.md).

## Troubleshooting

| Problem | Fix |
|---------|-----|
| `uv: command not found` | Install uv — see Quick Start above |
| `ModuleNotFoundError` | Run `uv sync`, and use `uv run uvicorn` instead of `uvicorn` directly |
| Port 8000 already in use | Use a different port: `uv run uvicorn main:app --reload --port 8001` |
| Docker: "Cannot connect to the Docker daemon" | Open Docker Desktop and wait for it to fully start |
| Windows: "running scripts is disabled" | Run `Set-ExecutionPolicy -ExecutionPolicy ByPass -Scope CurrentUser` in PowerShell |
